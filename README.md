# wine-ranking

A program to rank wines and view your past ratings.

## Features

- **Add a rating**: Enter a wine name and a rating from 0–10 (one decimal place, e.g. 7.5).
- **View all ratings**: See every wine you’ve rated, with score and date.
- **Delete a rating**: View the list and remove one by number (or press Enter to cancel).

Ratings are stored in `ratings.json` in this folder.

## Run

```bash
python wine_ranking.py
```

Requires Python 3.6+ (no extra packages).
